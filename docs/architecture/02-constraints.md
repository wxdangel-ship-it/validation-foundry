# 02 约束

## 状态

- 当前状态：项目级约束说明
- 来源依据：
  - `SPEC.md`
  - `AGENTS.md`
  - `docs/ARTIFACT_PROTOCOL.md`

## 全局约束

- 当前正式业务模块只有 `t01_pickup_point_validation`
- 不登录账号
- 不依赖用户长期驻场协助
- 禁止伪造坐标
- 禁止 silent failure
- 点击定位按钮后必须“至少等待 3 秒 + 页面稳定”才允许取点
- 输出坐标必须保持 `GCJ-02`

## 工程约束

- 文档与实现必须分离：
  - `modules/` 放文档
  - `src/validation_foundry/modules/` 放实现
- 默认禁止新增新的执行入口脚本
- 长期记忆必须写入仓库文档，而不是依赖对话历史
- 所有运行证据必须落在 `outputs/` 约定目录中

## 协作约束

- 子线程只处理一个清晰子问题
- 子线程必须写回指定文档
- 不允许多个线程无约束同时改同一批关键文件
- 阶段未通过 gate 前，不进入下一阶段
