# T04_IA

## 1. 页面关系

- P0 Demo Launcher 可直接进入三种模式样例
- P2 可进入 P4，也可通过弱入口跳到 P3
- P3 可进入 P4
- P4 可打开 P4-overlay，可结束 Trip 进入 P5

## 2. 页面优先级

- 第一优先级：P2、P3
- 第二优先级：P4、P4-overlay
- 第三优先级：P5
- 辅助优先级：P0

## 3. 卡片折叠策略

- 首轮默认全部展开
- 仅允许信息量较大的详情区做软折叠占位，不改变卡片顺序
- 固定操作区和离线卡不折叠

## 4. 状态优先级

- 阻塞态高于普通态
- Ready Gate 未满足时，高于“开始探索/开始前往入口”触发
- 安全抽屉开启时，地图仍作为背景可见
