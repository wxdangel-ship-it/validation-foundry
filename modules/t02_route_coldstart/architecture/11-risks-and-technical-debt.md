# 11 风险与技术债

## 当前主要风险

1. 两步路 `越野车` 标签存在混合标签误入，需要详情复核兜底。
2. 六只脚以搜索 `越野` 进入候选，召回更宽，误入率可能更高。
3. 官方导出轨迹能力可能不稳定，部分样本只能退化为 `screen_reconstructed`。
4. 真实可用的驾车越野路线密度可能不足，导致批量凑满 `20` 条需要扩大区域搜索。
5. 当前批量采集仍依赖真实设备 UI 操作，自动化程度有限。

## 当前可接受技术债

- 当前仍以真实设备采集为主，双源自动化助手尚未定版
- 当前详情语义判断仍带人工审核成分
- 当前入口只实现草案打包与 QA，不负责完整手机端自动导航

## 后续缓解方式

- 先完成双源首批 `20` 条闭环
- 再沉淀更稳定的页面采集助手
- 后续把多源适配抽离成独立 adapter 层
